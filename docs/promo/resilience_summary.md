# CWE Resilience Overview (for public readers)

**Threats & Defenses**
- Legal → Defensive licensing, open audits, opt‑in storage classes.
- Cyber → Signed builds, anycast hubs, verified IDs, rate limits.
- Sybil → SSI personhood, liveness checks, transparent delegation.
- Content → Creator signatures, encrypted sensitive areas, oversight councils.

**Sustainability**
Small transaction fee funds environmental assets that back the currency.

**Modularity**
Each layer—Ledger, Storage, DMF, Governance—is reusable beyond media.
